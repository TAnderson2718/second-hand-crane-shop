# 二手吊车小程序 (Taro版本)

基于原有HTML版本改造的微信小程序，使用Taro React框架开发，最大化复用了原有代码逻辑。

## 项目特色

### 🔄 代码复用
- **100%数据复用**: 完整迁移原HTML项目的模拟数据和业务逻辑
- **功能保持**: 保留了筛选、排序、收藏、拨打电话等所有原有功能
- **优化升级**: 使用React组件化开发，代码结构更清晰

### 📱 小程序优化
- **原生体验**: 完全适配微信小程序交互规范
- **性能优化**: 图片懒加载、下拉刷新、上拉加载
- **响应式设计**: 适配不同尺寸的手机屏幕

## 技术栈

- **框架**: Taro 3.6.20 + React 18
- **样式**: SCSS + 原生小程序样式
- **构建**: Webpack 5
- **开发语言**: JavaScript (ES6+)

## 项目结构

```
src/
├── app.js                 # 应用入口文件
├── app.config.js          # 应用配置文件
├── app.scss              # 全局样式文件
├── pages/                # 页面目录
│   ├── index/            # 首页
│   │   ├── index.jsx     # 首页组件
│   │   ├── index.config.js # 首页配置
│   │   └── index.scss    # 首页样式
│   ├── detail/           # 详情页
│   ├── post/             # 发布页
│   ├── profile/          # 个人中心
│   ├── favorites/        # 收藏页面
│   ├── posts/            # 我的发布
│   ├── intentions/       # 求购意向
│   └── report/           # 举报页面
├── services/             # 数据服务
│   └── data.js          # 数据管理和API
├── utils/                # 工具函数
│   └── index.js         # 通用工具函数
└── components/           # 通用组件
```

## 快速开始

### 环境要求
- Node.js >= 14.0.0
- npm >= 6.0.0
- 微信开发者工具

### 安装依赖
```bash
cd taro-miniprogram
npm install
```

### 开发运行
```bash
# 微信小程序
npm run dev:weapp

# H5 (用于调试)
npm run dev:h5
```

### 构建发布
```bash
# 构建微信小程序
npm run build:weapp

# 构建H5版本
npm run build:h5
```
